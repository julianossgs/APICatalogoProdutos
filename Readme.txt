Criação de Api

Escopo do projeto
 - Criar uma Web Api para um e-commerce de cerveja
 - Foi criado um serviço RESTful que permita que os aplicativos gerenciem o catálogo de produtos do e-commerce
   (criar,ler,editar e excluir os produtos e também consultar um produto específico)


Foi usado para a execução deste teste

- .Net 6.0

- AspNet Core 6.0

- Entity Framework Core 6

- Banco de dados MySql e ferramenta MySql Worbench

- Foi realizado o CRUD (Create - Update - Delete)

- Foram criado Endpoints usando os verbos HTTP: GET POST PUT DELETE

- Visual Studio 2022

- Usei Controllers e integração com Swagger

- Obs = a pasta contém 1 arquivo do SqlServer que pode ser desconsiderado
